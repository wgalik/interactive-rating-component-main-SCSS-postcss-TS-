# Interactive Rating Component

An interactive rating component built as a challenge from Frontend Mentor. It allows users to select a rating from 1 to 5 and displays a thank-you message after submitting the form.

## Demo

[Site]()

## Technologies

- **HTML5** – structure of the component
- **SCSS + PostCSS** – styling with variables, nesting, and gradients
- **TypeScript** – handling rating selection and view switching

## Features

- Select a rating from 1 to 5 using radio buttons
- Highlight the selected rating
- Thank-you screen displayed after form submission with the selected rating
- Responsive layout for different screen sizes
- Fully accessible (ARIA attributes, focus states)

## Usage

- Select a rating by clicking on one of the numbers from 1 to 5.
- Click **Submit** to see the thank-you screen.
- The selected rating will be displayed on the thank-you screen.

## Learning Points

- SCSS practice: variables, nesting, and gradients
- TypeScript DOM manipulation with proper typing
- Responsive and accessible component design
- Managing component state (rating form → thank-you view)
